## Component Hierarchy
  - Auth
  - User Detail
    - Shows teams
    - Shows races ordered by most recent date
  - Race Detail
    - Shows checkpoints
    - Shows competitors + bib number
  - Race Index
    - Shows all races for user
  - Checkpoint
    - Shows race name
    - Shows competitors + bib number
  - Team Detail
    - Shows all team members
  - Team Index [Bonus]
    - Shows all teams
